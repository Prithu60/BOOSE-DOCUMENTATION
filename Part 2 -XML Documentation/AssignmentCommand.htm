Class AssignmentCommand
Handles basic integer assignment statements in BOOSE.
Syntax
C#
    public class AssignmentCommand : Command
Remarks
Supported formats include: x = 5 x = y x = 2 * y This class evaluates the right-hand expression immediately and stores the result in the VariableStore.
