Class AppForBlock
Executes a complete for-loop block created by the parser. This class runs the loop body directly instead of using the standard BOOSE program counter logic.
Syntax
C#
    public class AppForBlock : Command
Remarks
This is used for block-based for loops in Part 2, where the parser collects the loop body and passes it here for execution.
