Property class
Stores all user-defined methods declared in a BOOSE program.
Syntax
C#
    public sealed class MethodStore
Remarks
This class follows the singleton pattern so that all parts of the interpreter access the same method definitions during execution. Methods can be added, retrieved, or cleared between program runs.
