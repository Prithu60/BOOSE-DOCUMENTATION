Class AppReset
Resets the BOOSE environment to its initial state.
Syntax
C#
    public class AppReset : CanvasCommand
Remarks
This command clears the canvas, resets the cursor position, and clears all stored variables and arrays. This is required for Part 2 to ensure repeatable runs.
