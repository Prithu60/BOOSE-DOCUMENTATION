Class AppWrite
Implements the WRITE command for BOOSE. Allows printing text, numbers, and boolean values onto the application canvas.
Syntax
C#
    public class AppWrite : Command
Remarks
This command supports: - String literals using quotes - Integer and real expressions - Boolean expressions and variables - String concatenation using +
