Interface IVariableStore
Defines a common interface for storing and retrieving variables used during BOOSE program execution.
Syntax
C#
    public interface IVariableStore
Remarks
This interface supports integer, real, and boolean variables, as well as one-dimensional integer and real arrays. Implementations are responsible for managing runtime state across commands, loops, and method calls.
