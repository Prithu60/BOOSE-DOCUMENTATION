Class AppArray
Implements the BOOSE array command.

This command allows declaration of integer and real arrays at runtime in BOOSE Part 2 programs.

Supported syntax:

  • array int nums 10
  • array real values 5

Syntax
C#
    public class AppArray : CommandThreeParameters
