Class AppWhile
Represents a WHILE loop block in BOOSE. Executes a list of commands repeatedly while the condition evaluates to true.
Syntax
C#
    public class AppWhile : Command
Remarks
The condition is evaluated before each iteration. A safety guard is used to prevent infinite loops.
