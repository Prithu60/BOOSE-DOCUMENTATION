Class ExpressionUtil
Utility class for evaluating numeric and boolean expressions at runtime.
Syntax
C#
    public static class ExpressionUtil
Remarks

This class is used throughout the BOOSE Part 2 interpreter to support unrestricted expressions involving variables, arithmetic operations, and boolean logic.

Identifiers within expressions are replaced with their runtime values before evaluation.
