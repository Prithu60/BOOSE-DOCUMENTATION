Documentation Preview
Class AppParser
Parses BOOSE source code and converts it into executable commands.
Syntax
C#
    public class AppParser
Remarks

This parser supports BOOSE Part 2 features including block structures such as if, while, for, and method.

It also supports implicit method definitions, unrestricted variable usage, and nested control blocks.
