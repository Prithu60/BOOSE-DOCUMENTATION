Class AppCommandFactory
Custom command factory for BOOSE Part 2. Responsible for creating the correct command object based on the command keyword.
Syntax
C#
    public class AppCommandFactory : CommandFactory
Remarks
This factory handles custom commands such as variables, arrays, methods, assignments, and drawing commands. Block commands (if, while, for, method) are handled separately by AppParser.
